Let's Learn
เซต (Set)
เซต ในวิชาคณิตศาสตร์ใช้กล่าวถึงกลุ่มของสิ่งต่างๆ และเมื่อกล่าวถึงกลุ่มใดแล้ว สามารถทราบได้แน่นอนว่าสิ่งใดอยู่ในกลุ่ม และสิ่งใดไม่อยู่ในกลุ่ม
Example เซตของชื่อวันในสัปดาห์ มีสมาชิก ได้แก่

จันทร์ อังคาร พุธ พฤหัสบดี ศุกร์ เสาร์ และอาทิตย์

การกำหนดเซตโดยใช้แผนภาพเวนน์-ออยเลอร์

แผนภาพเวนน์-ออยเลอร์ (Venn-Euler Diagram)

คือ แผนภาพที่นิยมใช้แสดงความสัมพันธ์ระหว่างเซต โดยนิยมใช้วงกลมหรือวงรีแทนเซต ซึ่งมีสมาชิกของเซตอยู่ภายในวงกลม และใช้รูปสี่เหลี่ยมผืนผ้าแทนเอกภพสัมพัทธ์
Example แผนภาพเวนน์-ออยเลอร์แสดงเซตของสมาชิกทั้งหมดที่เป็นสมาชิกของทั้งเซต A B และ C ซึ่งแทนด้วยสัญลักษณ์ A∩B∩C แทนด้วยพื้นที่ที่มองเห็นบริเวณที่ซ้อนทับกันระหว่างเซต A B และ C
** ในการเขียนเซตจะต้องกำหนดเขตที่บ่งบอกถึงขอบเขตของสิ่งที่จะพิจารณา เรียกเซตกำหนดเขตนี้ว่า เอกภพสัมพัทธ์ (Relative Universe) ซึ่งมักเขียนแทนด้วย U
การดำเนินการระหว่างเซต ( Operation on Set )
อินเตอร์เซกชัน( intersection )
เขียนแทนด้วย A ∩ B
มีบทนิยามว่า A ∩ B = { x | x ∈ A และ x ∈ B }
( x อยู่ในเซต A และเซต B )
Example

กำหนด A = { 1,2,3,4 } B = { 2,4,6,8 }

สร้างแผนภาพเวนน์-ออยเลอร์แสดงเซตของสมาชิกทั้งหมดที่เป็นสมาชิกของทั้งเซต A และ B ด้วยพื้นที่ที่มองเห็นบริเวณที่ซ้อนทับกันระหว่างเซต A และ B คือ 2 และ 4 จะเห็นว่า 2 และ 4 เป็นสมาชิกของทั้งเซต A และ B ดังภาพ เขียนแทนด้วย A ∩ B
ยูเนียน (union)
เขียนแทนด้วย A ∪ B
มีบทนิยามว่า A ∪ B = { x | x ∈ A หรือ x ∈ B }
( x อยู่ในเซต A หรือเซต B )
Example

กำหนด A = { 1,2,3,4 } B = { 2,4,6,8 }

สร้างแผนภาพเวนน์-ออยเลอร์แสดงเซตของสมาชิกทั้งหมดที่เป็นสมาชิกของเซต A หรือ B ด้วยพื้นที่ที่มองเห็นบริเวณของเซต A และ B คือ 1,2,3,4,6และ 8 จะเห็นว่า 1,2,3,4,6 และ 8 เป็นสมาชิกของเซต A หรือ B ดังภาพ เขียนแทนด้วย A ∪ B
คอมพลีเมนต์ (complement)
เขียนแทนด้วย A′
มีบทนิยามว่า A′ = { x | x ∈ U และ x ∉ A }
( x ไม่อยู่ในเซต A)
Example

กำหนด A = { 1,2,3,4 } B = { 2,4,6,8 }

สร้างแผนภาพเวนน์ - ออยเลอร์แสดงเซตของสมาชิกทั้งหมดที่ไม่เป็นสมาชิกของเซต A ด้วยพื้นที่ที่มองเห็นบริเวณที่ไม่ซ้อนทับกับเซต A และ B คือ 6 และ 8 จะเห็นว่า 6 และ 8 ไม่เป็นสมาชิกของเซต A ดังภาพ เขียนแทนด้วย A′
ผลต่างระหว่างเซต (difference of sets)
เขียนแทนด้วย A - B
มีบทนิยามว่า A - B = { x | x ∈ A และ x ∉ B }
( x อยู่ในเซต A แต่ไม่อยู่ในเซต B )
Example

กำหนด A = { 1,2,3,4 } B = { 2,4,6,8 }

สร้างแผนภาพเวนน์-ออยเลอร์แสดงเซตของสมาชิกทั้งหมดที่เป็นสมาชิกของเซต A แต่ไม่เป็นสมาชิกของเซต B ด้วยพื้นที่ที่มองเห็นบริเวณของเซต A แต่ไม่ซ้อนทับกับบริเวณของเซต B คือ 1 และ 3 จะเห็นว่า 1 และ 3 เป็นสมาชิกของเซต A หรือ B ดังภาพ เขียนแทนด้วย A - B
สมบัติการดำเนินการระหว่างเซต
สมบัติของอินเตอร์เซกชัน
กำหนดให้ A และ B เป็นสับเซตของเอกภพสัมพัทธ์ U แล้ว
  1. A ∩ A = A
  2. A ∩ Ø = Ø
  3. A ∩ U = A
  4. (A ∩ B) ⊂ A และ (B ∩ A) ⊂ B เมื่อ A = B
  5. A ∩ B = B ∩ A (สมบัติการสลับที่)
  6. A ∩ (B ∩ C) = (A ∩ B) ∩ C (สมบัติการเปลี่ยนกลุ่ม)
  7. A ⊂ B ก็ต่อเมื่อ A ∩ B = A
  8. ถ้า A ⊂ C และ B ⊂ C แล้ว (A ∩ B) ⊂ C
สมบัติของยูเนียน
กำหนดให้ A และ B เป็นสับเซตเอกภพสัมพัทธ์ U แล้ว
  1. A ∪ A = A
  2. A ∪ Ø = A
  3. A ∪ U = U
  4. A ⊂ (A ∪ B) และ B ⊂ (A ∪ B)
  5. A ∪ B = B ∪ A (สมบัติการสลับที่)
  6. A ∪ (B ∪ C) = (A ∪ B) ∪ C (สมบัติการเปลี่ยนกลุ่ม)
  7. A ⊂ B ก็ต่อเมื่อ A ∪ B = B
  8. ถ้า B ⊂ A และ C ⊂ A แล้ว (B ∪ C) ⊂ A
  9. ถ้า A ∪ B = Ø ก็ต่อเมื่อ A = Ø และ B = Ø
  10. A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C) (สมบัติการกระจาย)
  11. A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) (สมบัติการกระจาย)
สมบัติของคอมพลีเมนต์
กำหนดให้ A และ B เป็นสับเซตของเอกภพสัมพัทธ์ U แล้ว
  1. ( A′ )′ = A
  2. Ø′ = U และ U′ = Ø
  3. A ∩ A′ = Ø
  4. A ∪ A′ = U
  5. (A ∪ B)′ = A′ ∩ B′ (กฎของ เดอ มอร์กอง)
  6. (A ∩ B)′ = A′ ∪ B′ (กฎของ เดอ มอร์กอง)
  7. A ⊂ B ก็ต่อเมื่อ B′ ⊂ A′
  8. A ∩ B = Ø ก็ต่อเมื่อ A ⊂ B′
สมบัติของผลต่าง
กำหนดให้ A และ B เป็นสับเซตเอกภพสัมพัทธ์ U แล้ว
  1. A - B ⊂ A
  2. A - B = A ก็ต่อเมื่อ A ∩ B = Ø
  3. A - B = Ø ก็ต่อเมื่อ A ⊂ B
  4. A - A = Ø
  5. A - B = A ∩ B′
  6. A - (B ∪ C) = (A - B) ∩ (A - C)
  7. A - (B ∩ C) = (A - B) ∪ (A - C)
  8. (A ∩ B) - C = (A - C) ∩ (B - C)
  9. (A ∪ B) - C = (A - C) ∪ (B - C)
  10. A - Ø = A และ Ø - A = Ø
  11. A′ - B′ = B - A
